# Problem ↔ Solution Explorer

현대 산업의 **문제(Problem) ↔ 솔루션(Solution)** 관계를 탐색하는 인터랙티브 도구

**[→ 라이브 데모 보기](https://m1zz.github.io/industry-explorer/)**

## 세 가지 뷰

- 🏭 **순방향 트리** — 도메인 → 세부산업 → 솔루션 → 문제
- 🔴 **역방향 트리** — 문제 → 솔루션 → 도메인
- 🕸️ **네트워크 그래프** — 전체 구조 시각화
